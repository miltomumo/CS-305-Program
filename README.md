# CS-305-Program
My client for this project was Artemis Financial, a financial company that focuses on providing custom customer budgeting. The goal of the project was to help Artemis financial modernize their operations and improve security. For this specific software, the company needed me to analyze existing vulnerabilities and apply secure comunication mechanisms.
  I started off by creating a certificate that ensured secure comunications between servers, and I created a hashing algorightm to encrypt things like passwords and personal sensitive information. Additionally I manually reviewed the code and automated tools to ensure that I didn't introduce new vulnerabilities.
  The most challenging part of the vulnerability asssesment was reading and interpreting the report, I had to learn the difference between a false positive and how to supress it so the report only flagged the true positives that could be fixed.
  I increased the security of Artemis Financial by implementing the SHA-256 encryotion algorithm to protect sensitive data, as well as a certificate and enforcing HTPPS to ensure trusted and secure communications. Finally, I ran an OWASP dependency check to ensure I never introduced new vulnerabilities. In the future, I will continue utilizing Dependency checks for automated vulnerability assesment and I will continue to manually review the code.
  I made sure that the application remained fuctional and secure by compiling the code as I worked to avoid creating errors. I also checked the HTTPS endpoint in a browser, and validated checksum outputs.
  Some resources I will apply in future assignments are the OWASP Dependency-Check for static analysis, because it provides detailed security vulnerabilities with ways of fixing them. Checksum verification so I can test for integrity. FOr future employers, I would show my ability to review an existing code via a static dependency-check and a checksum verification. Additionally, I would show how I was able to refactore the code to inprove the security by implementing hashing, HTTPS and a certificate.
